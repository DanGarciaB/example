¿Que es el desarrollo web? DANIEL GARCIA

Es el trabajo que tiene lugar en segundo plano y que permite
que una web tenga una apariencia impecable, un funcionamiento rápido
y un buen desempeño para permitir la mejor experiencia de usuario.

En el campo del desarrollo web, las tecnologías se dividen tecnicamente en dos,
Backend y Frontend, ¿Que es esto?, es muy sencillo de explicar:

FronEnd

El frontend son todas aquellas tecnologías que corren del lado del cliente, es decir,
todas aquellas tecnologías que corren del lado del navegador web, generalizandose mas que nada en tres lenguajes.

  • HTML
  • CSS
  • JavaScript

En pocas palabras, el FrontEnd, es el que se encargará de dejar bonita la página, en ver que los datos
se muestren de manera comoda para el usuario, de que la interacción que realice sea llamativa y en la estética del sitio.

BackEnd

El programador backend es aquel que se encuentra del lado del servidor, es decir, esta persona se encarga
de lenguajes como:

  • PHP
  • Python
  • .Net
  • Java
